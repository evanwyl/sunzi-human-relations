# Case bank

用这个文件沉淀“局型”，不是只堆故事。

每个案例都尽量压成同一结构，方便后续复用、扩写、改写。

## 统一结构

- 场景：职场 / 朋友 / 情侣 / 暧昧
- 局型：一句话命名这个局
- 用户表面问题：用户嘴上在问什么
- 底层冲突：真正卡住的矛盾是什么
- 主动权：谁更急、谁更有退路、谁更依赖
- 关键信号：哪些细节说明这是这个局
- 常见误判：用户最容易怎么误读
- 推荐兵法：1 到 2 条《孙子兵法》原则 + 1 条《三十六计》
- 最优动作：低成本、高信息量、可执行
- 禁忌动作：最容易把局打坏的动作
- 观察点：做完后应该观察什么反馈

## 职场

### 1. 领导表面器重，实际持续甩锅
- 场景：职场
- 局型：捧高你，然后让你接高风险低收益的活
- 用户表面问题：领导总是信任我，把难活都给我，我要不要拒绝
- 底层冲突：不是信任，而是风险转移与角色固化
- 主动权：领导有分配权，用户有执行力但退路不足
- 关键信号：夸你靠谱、关键时刻让你兜底、出事时功劳模糊责任清晰
- 常见误判：把“被需要”误判成“被重用”
- 推荐兵法：上兵伐谋，其次伐交 + 善战者，求之于势，不责于人 + 釜底抽薪
- 最优动作：把工作量、风险、支持条件和优先级显性化，逼分工机制浮出水面
- 禁忌动作：情绪化抱怨、当众翻脸、继续硬扛
- 观察点：对方是否愿意重新分工、给资源、明确责任归属

### 2. 同级同事抢功，平时又装合作
- 场景：职场
- 局型：表面协作，实则抢叙事权
- 用户表面问题：同事总是在领导面前把我的成果说成团队成果，怎么办
- 底层冲突：不是合作问题，是成果归属和话语权争夺
- 主动权：对方更会讲故事，用户更像干活的人
- 关键信号：会前不吭声，会后抢总结；口头感谢很多，署名模糊
- 常见误判：以为只要多干，领导自然会看见
- 推荐兵法：知彼知己，百战不殆 + 计利以听，乃为之势，以佐其外 + 擒贼擒王
- 最优动作：提前抢节点汇报、书面留痕、锁定关键成果归属
- 禁忌动作：私下委屈、不留证据、最后时刻才解释
- 观察点：领导的信息来源是否开始更多直接经过你

### 3. 领导冷处理你，但不明说不满
- 场景：职场
- 局型：不撕破脸，但持续边缘化
- 用户表面问题：领导最近不怎么理我，是不是对我有意见
- 底层冲突：不是一句态度问题，而是资源分配和信任等级在下滑
- 主动权：领导掌控资源，用户信息不足
- 关键信号：会少叫你、信息少同步、关键任务绕开你
- 常见误判：急着解释、急着表忠心
- 推荐兵法：先为不可胜，以待敌之可胜 + 隔岸观火
- 最优动作：先补信息，不先摊牌；观察是否是阶段性还是结构性降权
- 禁忌动作：带情绪追问“我是不是做错了什么”
- 观察点：未来两到三次项目分配是否持续被排除

## 朋友

### 4. 对方长期索取，偶尔示好续命
- 场景：朋友
- 局型：平时消耗你，快断了再给点甜头
- 用户表面问题：这个朋友平时都不怎么管我，但偶尔对我很好，我要不要继续
- 底层冲突：关系维系靠你的投入，对方靠间歇性示好防止你退出
- 主动权：你投入更多，对方节奏更松
- 关键信号：有事找你，没事消失；你要撤时又突然热情
- 常见误判：把偶尔示好当成关系回暖
- 推荐兵法：兵贵胜，不贵久 + 以逸待劳
- 最优动作：减少主动补位，看关系是否还能自发回流
- 禁忌动作：因为一次示好又恢复全部投入
- 观察点：你停手后，对方是稳定补位还是再次沉下去

### 5. 表面关系不错，实际总被轻慢冒犯
- 场景：朋友
- 局型：边界被侵蚀，但包装成玩笑
- 用户表面问题：朋友老开我不舒服的玩笑，我是不是太敏感
- 底层冲突：不是玩笑尺度问题，而是尊重与边界问题
- 主动权：对方占口头上风，用户怕翻脸
- 关键信号：你不舒服后，对方说你开不起玩笑；类似行为反复发生
- 常见误判：把反复冒犯解释成性格直
- 推荐兵法：善战者，求之于势，不责于人 + 关门捉贼
- 最优动作：一次明确划线，不讨论感受对错，只定边界和后果
- 禁忌动作：一边忍一边积怨，最后一次性爆炸
- 观察点：对方是收手，还是继续试探边界

## 情侣

### 6. 吵架后对方长期沉默，占节奏权
- 场景：情侣
- 局型：不用吵赢，而用沉默让你先低头
- 用户表面问题：每次一吵架他就不说话，我该不该去哄
- 底层冲突：不是沟通风格差异，而是节奏权和修复成本分配失衡
- 主动权：沉默的一方短期更占主动，着急修复的一方更被动
- 关键信号：一冲突就消失、等你先找、复合后不处理根因
- 常见误判：以为只要先低头，关系就会变好
- 推荐兵法：先为不可胜，以待敌之可胜 + 欲擒故纵
- 最优动作：停止无底线追修复，明确修复窗口和沟通条件
- 禁忌动作：连发长消息、不断解释、把自己全部摊开
- 观察点：对方是否愿意在明确边界下回来沟通

### 7. 对方反复承诺会改，但模式不变
- 场景：情侣
- 局型：语言修复快，结构改变慢
- 用户表面问题：他说会改，我要不要再给一次机会
- 底层冲突：不是态度问题，而是有没有真实行为成本与改变结构
- 主动权：谁更怕失去，谁更容易让步
- 关键信号：每次都认错、每次都复发、没有具体改变措施
- 常见误判：把会说当成会做
- 推荐兵法：知彼知己，百战不殆 + 兵贵胜，不贵久 + 打草惊蛇
- 最优动作：不给空头承诺加分，只看一次具体行为验证
- 禁忌动作：继续按感觉续命
- 观察点：对方是否真的承担了改变成本

## 暧昧

### 8. 对方持续给回应，但始终不推进关系
- 场景：暧昧
- 局型：给注意力，不给定义
- 用户表面问题：他明明会回我、也会找我，但就是不提关系，什么意思
- 底层冲突：他可能享受关系收益，但不愿承担关系成本
- 主动权：定义关系的人更被动，享受模糊的人更主动
- 关键信号：聊天不冷、现实推进弱、关键节点总模糊带过
- 常见误判：把有回应当成有承诺
- 推荐兵法：上兵伐谋，其次伐交 + 打草惊蛇
- 最优动作：用轻量试探看他愿不愿意付出更具体投入
- 禁忌动作：一上来摊牌、控诉、逼承诺
- 观察点：对方是补动作，还是继续维持模糊

### 9. 对方忽冷忽热，但从不真正失联
- 场景：暧昧
- 局型：放线钓着，防止你彻底退出
- 用户表面问题：他到底喜不喜欢我
- 底层冲突：重点不是喜不喜欢，而是投入意愿与关系排序
- 主动权：对方靠节奏变化控制你的期待
- 关键信号：冷一阵，热一下；总能在你快放下时出现
- 常见误判：把“回来找你”当成认真回流
- 推荐兵法：避实而击虚 + 欲擒故纵 + 走为上计
- 最优动作：停掉高投入，观察他是否会主动补位和推进
- 禁忌动作：每次回暖就重启全部情绪投入
- 观察点：对方是否会在你收手后给出更清晰动作

### 10. 前任回头联系，但动机不明
- 场景：暧昧 / 前任
- 局型：情绪回流，不等于关系回流
- 用户表面问题：前任突然找我，是不是想复合
- 底层冲突：对方可能是试探、补偿、寂寞、后悔，未必是要重建关系
- 主动权：谁先重新投入，谁先暴露底牌
- 关键信号：寒暄很多、具体承担很少；试探情绪，不谈未来
- 常见误判：把联系本身当成复合意图
- 推荐兵法：知彼知己，百战不殆 + 打草惊蛇
- 最优动作：低暴露回应，先识别动机，不先开旧账也不先交底
- 禁忌动作：一上来倾倒情绪或直接谈复合
- 观察点：对方后续是否持续、具体、承担性地靠近
