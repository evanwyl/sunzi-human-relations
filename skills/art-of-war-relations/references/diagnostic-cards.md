# Diagnostic cards

先拆局，再给建议。每次至少在内部判断下面这些卡位，避免只给情绪安慰。

## 1. 这是什么局
- 场景：职场 / 朋友 / 情侣 / 暧昧
- 用户要什么：看局 / 建议 / 回复话术 / 要不要继续投入 / 判断对方意图
- 当前阶段：刚起冲突 / 反复拉扯 / 快到决断点 / 已经烂尾

## 2. 表面冲突 vs 底层冲突
- 表面冲突：用户嘴上在抱怨什么
- 底层冲突：权力 / 资源 / 信任 / 节奏 / 边界 / 角色 / 需求错位

## 3. 主动权卡
必须判断，不要跳过。

- 谁更急
- 谁更依赖
- 谁更怕失去
- 谁更有替代选项
- 谁更能承受拖延

结论上要能说清：
- 主动权当前在谁手里
- 用户是输在信息差、节奏差，还是筹码差

## 4. 筹码卡
- 用户手里有什么：证据、能力、位置、情感价值、稀缺性、选择权
- 对方手里有什么：资源、话语权、模糊空间、时间优势、社会位置
- 哪边在消耗，哪边在等待

## 5. 真实诉求卡
- 用户真正想要的是：被爱、被尊重、被选中、公平、确定性、补偿、体面退出，还是反败为胜
- 对方大概率要的是：低成本维持、继续模糊、转移责任、保留好处、控制节奏，还是认真修复

## 6. 信号 vs 噪音
区分：
- 什么是一次性情绪
- 什么是重复模式
- 什么是有效投入
- 什么只是嘴上表态

原则：重复行为 > 单次表态；实际投入 > 情绪语言；结构变化 > 一时态度

## 7. 最优动作卡
优先级固定：
1. 低成本获取信息
2. 先稳住自身位置
3. 再决定推进、观望、试探、换打法或退出

如果建议一个动作，必须能回答：
- 这个动作想验证什么
- 做完后看什么反馈
- 如果反馈不好，下一步是什么

## 8. 止损卡
要敢于判断：
- 什么时候继续投入已经不划算
- 什么时候该减少投入
- 什么时候该结束这局

退出不是失败，是对烂局停止加注。
